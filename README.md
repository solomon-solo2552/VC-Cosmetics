# VC-Cosmetics

This is a **Python Django Project** for managing a cosmetics store.

## Features
- User registration and login
- Product management
- Cart and checkout system
- Admin dashboard

## Tech Stack
- Python 3.x
- Django
- SQLite (or PostgreSQL)

## How to Run
Clone the repo and follow standard Django setup:

```bash
git clone https://github.com/solomon-solo2552/VC-Cosmetics.git
cd VC-Cosmetics
python -m venv venv
venv\Scripts\activate   # Windows
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
